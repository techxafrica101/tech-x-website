# Tech-X Official Website

A professional marketing website for Tech-X - a Zambian technology and innovation company building smart digital solutions for Africa.

## Website Structure

### **Homepage (index.html)**
- **Hero Section**: Auto-typing live project showcase
- **Services**: Five core service offerings
- **Why Tech-X**: Trust-building differentiators
- **Projects**: Current and upcoming projects with status indicators
- **How We Work**: 4-step professional process
- **Current Focus**: StatFlow platform spotlight
- **About**: Vision statement and social proof
- **Latest Update**: Preview of most recent update
- **Contact**: Professional contact form and details

### **Updates Page (updates.html)**
- **Strategic Content Hub**: Professional updates and insights
- **Three Content Categories**:
  - **Building**: Technical progress and development updates
  - **Insight**: Industry observations and learnings
  - **Vision**: Strategic thinking and company direction
- **Professional Layout**: Clean cards with date, category, and content
- **SEO Optimized**: Structured content for search visibility

## Key Features

- **Professional Design**: Investor-ready, calm, and confident aesthetic
- **Trust Building**: Multiple credibility signals throughout
- **Mobile Responsive**: Optimized for all device sizes
- **Performance Focused**: Fast loading with smooth animations
- **Strategic CTAs**: Clear conversion paths for different user types

## Technologies Used

- HTML5

## Deployment

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select source branch (main/master)
4. Your site will be available at `https://yourusername.github.io/repository-name`

### Netlify
1. Drag and drop the project folder to [Netlify](https://netlify.com)
2. Or connect your GitHub repository for automatic deployments

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy

## Customization

- **Colors**: Modify the Tailwind config in the `<script>` tag
- **Content**: Update text content directly in the HTML
- **Images**: Add your own images to enhance the design
- **Contact Form**: Integrate with a backend service or form handler
