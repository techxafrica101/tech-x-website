# Tech-X Official Website

A clean, modern marketing website for Tech-X - a technology and innovation company focused on building smart digital solutions for Africa.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices
- **Modern UI**: Clean, minimalist design with Tailwind CSS
- **Single Page Application**: All sections on one page with smooth scrolling navigation
- **Professional Branding**: Consistent color scheme and typography
- **Interactive Elements**: Hover effects, smooth transitions, and mobile menu

## Sections

1. **Hero Section**: Main headline and call-to-action
2. **Services**: Five key service offerings with icons
3. **Projects**: Current and upcoming projects with status badges
4. **About**: Company overview and mission
5. **Contact**: Contact form and company details
6. **Footer**: Social links and copyright

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome Icons
- Vanilla JavaScript for interactions

## Deployment

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select source branch (main/master)
4. Your site will be available at `https://yourusername.github.io/repository-name`

### Netlify
1. Drag and drop the project folder to [Netlify](https://netlify.com)
2. Or connect your GitHub repository for automatic deployments

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy

## Customization

- **Colors**: Modify the Tailwind config in the `<script>` tag
- **Content**: Update text content directly in the HTML
- **Images**: Add your own images to enhance the design
- **Contact Form**: Integrate with a backend service or form handler
